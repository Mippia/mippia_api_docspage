# Dataset Management

Create and manage custom datasets for plagiarism detection.

> Due to copyright issues, custom dataset creation are available by request only. Please [contact us](https://platform.mippia.com/contact) for access.

## Dataset Creation

Create a new dataset with your music collection. 
```
POST /v1/dataset
```

:::{note}
This endpoint is currently in development. [Contact us](https://platform.mippia.com/contact) for beta access.
:::

## Dataset Add

Add tracks to an existing dataset.
```
POST /v1/dataset/{:dataset_id}/music
```

:::{note}
This endpoint is currently in development. [Contact us](https://platform.mippia.com/contact) for beta access.
:::

## Pricing

- **Free tier**: Up to 1,000 tracks per dataset
- **Processing time**: ~2 minutes per track
- **Enterprise**: For datasets exceeding 1,000 tracks, [Contact us](https://platform.mippia.com/contact) for access.
