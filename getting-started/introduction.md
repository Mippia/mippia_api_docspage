# Introduction

MIPPIA API provides music analysis capabilities for detecting AI-generated music, identifying plagiarism, and analyzing lyric similarity.

## What You Can Do

- **AI Detection** – Identify whether a track was AI-generated or human-created
- **Plagiarism Detection** – Compare tracks against custom datasets
- **Lyric Analysis** – Detect semantic similarity between lyrics in 100+ languages

## Key Features

- Fast processing (30 seconds to 4 minutes depending on API)
- Multiple accuracy tiers (Lite, Standard, Pro)
- Webhook support for async results
- Multi-language lyric analysis
- Custom dataset management
